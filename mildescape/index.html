MILD ESCAPE ARCHIVE


「MILD ESCAPE」はTESSHI-e氏によって制作されたFlash製脱出ゲームのシリーズです。
このウェブサイトは、ゲームの公開停止とFlashの終了に伴いプレイ不可能となったMILD ESCAPEのアーカイブです。
Flashの再生には、Ruffleに、日本語フォントや一部機能の修正を加えたKabanFriends/ruffleを使用しています。

This website is an archive of MILD ESCAPE, a series of point-and-click escape flash games, created by TESSHIE-e.
The text below is a list of games that are no longer public due to the Flash Player End of Support, restored in playable forms.
For running the Flash games, it uses Ruffle with custom modifications such as adding Japanese font and other minor fixes.
